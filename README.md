Installation
====================
	```
		pip install -r requirements.txt
	```


Setup & Running
====================

## Setup:

  Clone:
  	```
  		git clone https://github.com/Selectaman/LearningDjango.git
  	```
## Running:

  	```
		python manage.py makemigrations
		python manage.py migrate
		python manage.py runserver
  	```
